Simple twitter feed shortcode using Twitter API 1.1

== Description ==

Simple twitter feed shortcode using Twitter API 1.1 - takes parameters to control query.

Uses https://github.com/micc83/Twitter-API-1.1-Client-for-Wordpress for OAuth and query.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add [twitterfeed] shortcode anywhere along with desired parameters
4. Style as needed. Multiple css hooks are provided
